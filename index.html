Featured Project: Text Analysis

In this digital humanities project, I explored the first two chapters of The Hound of the Baskervilles using XML markup and Voyant Tools. I tagged elements like characters and objects in the text, then used Voyant to analyze patterns across a mystery novel corpus.

This project helped me see how digital tools can reveal hidden structures in storytelling and provided a unique way to connect close reading with data visualization.
